Luno-PHP

Go to Repository

Luno provides a fully featured User Authentication system. We give you the flexibilty to build your own login & registration system atop of our API's, or use our pre-provided templates to save on coding time!

This library makes it exceptionally easy to get started immediately with PHP when using Luno.

Installation

Luno-php uses composer to manage dependencies. We can require the library by:

composer require duffleman/lunophp

After that, you can include your autoloader into any PHP file and use Luno-PHP immediately.

Usage

Code Examples 
<?php

use Duffleman\Luno\LunoRequester;

require_once(__DIR__ . '/../vendor/autoload.php');

$luno = new LunoRequester([
    'key'     => '<YOUR_LUNO_KEY>',
    'secret'  => '<YOUR_LUNO_SECRET>',
    'timeout' => 10000
]);

$user = $luno->users->create([
    'username' => 'Duffleman',
    'name'     => 'James Duffleman',
    'email'    => 'james@duffleman.co.uk',
    'password' => 'myPassword',
    'profile'  => [
        'attribute' => 'value'
    ]
]);

Easy :)

We recommend using environment variables to store the key and secret.
